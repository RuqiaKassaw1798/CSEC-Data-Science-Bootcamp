# CSEC Data Science bootcamp: Task 1 Project Suite
This folder contains my comprehensive solutions for CSEC Data Science bootcamp Daily assignment task. It spans from basic syntax and algorithmic logic to functional programming and game state management.
## 📁 Project Structure
The project is organized into three distinct levels of complexity:
### Level 1: Core Logic & Syntax
 * Prime Number Checker: Determines primality using a for-else loop with early break optimization.
 * Factorial Calculator: Implements the accumulator pattern to calculate n! for non-negative integers.
 * String Reversal: Uses Python’s extended slicing ([::-1]) for efficient character manipulation.
### Level 2: Data Manipulation
 * Fibonacci Sequence: Generates n terms using parallel assignment (a, b = b, a + b) and clean comma-formatting.
 * Vowel Counter: Uses membership testing to count occurrences in a string while maintaining case-insensitivity.
 * Max Number Finder: A CLI tool that parses a list of space-separated strings and identifies the maximum value.
### Level 3: Functional Systems
 * Modular Calculator: A functional arithmetic tool demonstrating DRY principles and error handling for division.
 * Tic-Tac-Toe Game: A full 2-player CLI game utilizing a 2D list (grid) and a continuous game loop.
## 🛠️ Technical Skills Demonstrated
 * Algorithmic Thinking: Logic optimization (early breaks, O(n) searches).
 * Data Structures: Strings, Lists, and Nested Lists (Matrices).
 * Clean Code: F-string formatting, modular functions, and descriptive naming.
 * Error Handling: Managing invalid inputs and mathematical errors (DivisionByZero).
